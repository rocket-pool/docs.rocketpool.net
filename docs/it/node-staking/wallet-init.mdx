# Creazione di un Nuovo Wallet

Se i servizi Smartnode sono attivi e funzionanti, il passo successivo è configurare un wallet Ethereum per il tuo nodo.
Si tratta semplicemente di un indirizzo Execution Layer che conterrà i fondi del tuo nodo - lo utilizzerà per inviare ETH al tuo minipool quando inizi lo staking, pagare il gas durante varie transazioni e altre operazioni.

Hai la possibilità di creare un wallet completamente nuovo da zero, oppure importare / recuperare un wallet o indirizzo esistente.
Scegli l'opzione che preferisci dalle schede qui sotto.

<div className="p-3">
  <Tabs>
    <Tab label="Creazione di un Nuovo Wallet">
      Il modo più comune per eseguire un nodo è creare un nuovo indirizzo Ethereum dedicato al nodo.
      La CLI Smartnode fornisce un modo semplice per farlo:

      ```shell
      rocketpool wallet init
      ```

      Ti verrà prima richiesta una password per proteggere la chiave privata del tuo wallet.
      Successivamente, ti verrà presentato il **mnemonic unico di 24 parole** per il tuo nuovo wallet.
      Questa è la **frase di recupero** per il tuo wallet.
      Se dovessi perdere la tua macchina, puoi utilizzare questa frase per rigenerare il tuo wallet e ripristinare tutti i validator Consensus ad esso collegati.

      <p className="rspress-directive warning">
        <p className="rspress-directive-title">NOTA</p>
        È **essenziale** che tu scriva questo mnemonic perché questa è l'unica volta che ti verrà mostrato, ma conservalo in un luogo sicuro.
        Chiunque abbia questa frase può ottenere il controllo del tuo wallet.
      </p>

      Una volta confermata la tua frase mnemonic, ti verrà presentato l'indirizzo Ethereum unico per il wallet del tuo nodo.
    </Tab>
    <Tab label="Recupero o Importazione di un Wallet Esistente">
      Una volta completato l'inserimento della frase mnemonic, vedrai un messaggio di successo come il seguente:
      "The node wallet was successfully initialized.
        Node account: {LungaStringaDiLettereENumeri}"
      Questo è l'indirizzo pubblico a cui dovrai inviare ETH in "Preparazione del tuo Nodo per l'Operatività". Puoi anche verificarlo digitando "rocketpool wallet status".

      Se hai già un wallet nodo che hai creato in una precedente installazione di Smartnode, o se vuoi importare un indirizzo che hai creato altrove (come ad esempio con **Allnodes**), segui le istruzioni nella guida [Importazione di un Wallet Esistente per il tuo Nodo](./recovering-rp).

      Torna qui quando hai finito.
    </Tab>

  </Tabs>
</div>

::: warning NOTA
Nella versione attuale, Rocket Pool deve avere accesso alla chiave privata del tuo wallet per eseguire le sue funzioni automatiche.
**Ciò significa che la chiave privata esisterà in un file sulla tua macchina.**
Se un attaccante riesce ad ottenere accesso alla tua macchina, potrebbe accedere al wallet del tuo nodo e rubare tutti i token in esso contenuti!
Assicurati di aver seguito la guida di sicurezza nella sezione [Protezione del tuo Nodo](./securing-your-node) prima di continuare e di essere soddisfatto della tua postura di sicurezza.
:::

      La chiave privata del tuo wallet sarà archiviata in un file situato in `~/.rocketpool/data/wallet`.

      La password utilizzata per crittografare la chiave del tuo wallet sarà archiviata in `~/.rocketpool/data/password`.
      Questo è il file che deve essere protetto in ogni momento.
    </Tab>

      La chiave privata del tuo wallet sarà archiviata in un file situato in `/srv/rocketpool/data/wallet`.

      La password utilizzata per crittografare la chiave del tuo wallet sarà archiviata in `/srv/rocketpool/data/password`.
      Questo è il file che deve essere protetto in ogni momento.
