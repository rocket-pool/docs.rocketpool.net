# Panoramica

In questa sezione, imparerai come installare lo stack Smartnode di Rocket Pool sulla tua macchina nodo.
Ci sono diversi tipi di installazione; imparerai quale è giusto per te e seguirai il processo di installazione per esso.

## Prerequisiti

Prima di tuffarti in questa sezione, assicurati di aver:

- Scelto una macchina nodo (hardware locale o server virtuale)
- Configurato il sistema operativo su di essa
- Configurato un sistema di controllo remoto come [SSH](../ssh)
- Protetto seguendo la guida [Protezione del tuo nodo](../securing-your-node)

## Guide

[Scelta dei tuoi Client ETH](../eth-clients) presenta le varie opzioni per i client Ethereum - sia sul livello Execution (precedentemente ETH1) che sul livello Consensus (precedentemente ETH2).
Avrai bisogno di uno di ciascuno per eseguire il tuo nodo, e questa sezione ti aiuterà a scegliere quali vorresti eseguire.

[Selezione di una Modalità Rocket Pool](../install-modes) ti mostrerà le diverse modalità con cui può essere installato lo Smartnode di Rocket Pool e ti aiuterà a scegliere la modalità migliore per te.

[Creazione di un Nodo Rocket Pool Standard con Docker](../docker) ti guiderà attraverso il processo di installazione per **Modalità Docker** e **Modalità Hybrid**.

[Creazione di un Nodo Rocket Pool Nativo senza Docker](../native.mdx) ti guiderà attraverso il processo di installazione per **Modalità Native**.
