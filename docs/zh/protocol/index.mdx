# 简介

Rocket Pool 是首个同类型的 ETH 权益证明协议，旨在实现社区所有、去中心化、无需信任，并与以太坊质押兼容。该协议于 2016 年末首次构思，并于 2021 年 10 月正式上线。

Rocket Pool 旨在服务两类主要用户群体：一类是希望通过 [rETH 代币化质押](https://medium.com/rocket-pool/rocket-pool-2-5-tokenised-staking-48601d52d924#92b0) 参与质押的用户，最低只需 **0.01 ETH**；另一类是希望质押 ETH 并在网络中运行节点的用户，通过赚取佣金获得比协议外质押更高的投资回报率。

如需了解构成协议的这两类群体的更多信息，我们强烈建议 [阅读系列讲解文章的第一篇](https://medium.com/rocket-pool/rocket-pool-staking-protocol-part-1-8be4859e5fbd)，其中详细介绍了用户如何参与，无论是通过代币化质押还是在协议中运行节点。

协议的核心理念是确保网络不受任何单一方的控制。这一原则与以太坊和 ETH 本身直接相关，也是 Rocket Pool 发展过程中每个阶段所秉持的思维方式。

Rocket Pool 致力于体现以太坊和 DeFi 的核心精神，特别是非托管、无需信任的特性，使真正的自我主权得以蓬勃发展。

这就是为什么创建这个 ETH 质押基础协议层如此重要，尤其是考虑到绝大多数参与者要么缺乏运行节点的技术能力，要么没有持有 32 ETH 的财务实力。
