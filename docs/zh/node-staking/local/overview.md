# 概述

本节旨在帮助您了解准备本地机器以在家中用作 Rocket Pool 节点的过程。

## 指南

[选择质押硬件](./hardware)展示了各种可行的节点操作硬件配置,希望能在您选择要购买的设备时为您提供灵感。

[准备 PC、Mini-PC 或 NUC](./prepare-pc)将引导您完成在机器上安装和配置基本操作系统的基本步骤,为其作为节点的生命做准备。

[准备 Mac](./prepare-mac)与上述相同,但适用于 macOS 系统。

[Secure Shell (SSH) 简介](../ssh)向您展示如何使用 **SSH**,这是一种流行的远程访问工具,用于从不同机器(例如笔记本电脑或台式机)远程和安全地使用您的节点机器,这样您就不必通过连接到它的显示器和键盘直接与节点交互。
