# Saturn 0 升级

欢迎来到 Saturn 0！此升级旨在短期内改善 RPL 代币经济学以解锁协议增长。如果您是回归或未来的节点运营者，本页将作为 Rocket Pool 协议的一些新的和令人兴奋的变化的介绍。

## Saturn 0 中的纯 ETH minipool

- 新 minipool 的 RPL 要求已成为过去！Saturn 0 中的 minipool 不需要 RPL 质押即可部署。
- 新 minipool 的 RPL 要求已成为过去！Saturn 0 中的 minipool 不需要 RPL 质押即可部署。
  节点运营者现在可以以 8 ETH 的前期成本创建 minipool，佣金为 10%。此外，节点运营者可以选择性地质押 RPL 以获得额外奖励并访问[协议 DAO](/zh/legacy/houston/pdao#the-protocol-dao-pdao) 内的投票权。

- 任何规模的 RPL 质押节点都有资格获得 RPL 发行奖励。这为 10% 抵押品悬崖下的节点运营者解锁了 RPL 奖励，这些运营者以前没有资格获得 RPL 发行奖励。

- 这意味着借入 ETH 的 0-15% RPL 质押节点可获得 RPL 的最高 APY。节点仍然可以在 15% 抵押品以上获得 RPL 收益，但收益率递减。
  请随时查看 [RPIP-30](https://rpips.rocketpool.net/RPIPs/RPIP-30#specification) 中如何定义超过 15% 借入 ETH 的 RPL 发行。

## 新的佣金结构

我们很高兴地宣布，新推出的 Saturn 0 minipool 将具有 10% 至 14% 的佣金结构。

- 没有任何质押 RPL 的节点将获得 10% 的佣金（5% 来自 minipool 合约，5% 来自动态佣金提升）。
- 质押了 RPL 价值 >= 借入 ETH 的 10% 的节点可额外获得高达 4% 的动态佣金提升。

请注意，在 Saturn 1 发布后的第 5 个奖励快照中，动态佣金提升将降低至 0%（仅保留 5% 的 minipool 合约佣金）。这有助于增加迁移到 megapool 的收益优势。

::: warning 警告
您的节点必须选择加入[平滑池](/zh/node-staking/fee-distrib-sp#the-smoothing-pool)才能从动态佣金提升中受益。如果您尚未选择加入，Smart Node 将在存入 minipool 时通知您选择加入平滑池。_如果您的节点不在平滑池中，您的新 minipool 将错过额外的收益！！_
:::

## 节点运营者指南

如果您有兴趣运行 Rocket Pool 节点以创建自己的 Ethereum 验证器，请查看我们的[节点运营者指南](/zh/node-staking/responsibilities)。
这是一个非常全面的指南，涵盖了节点操作涉及的所有考虑因素，包括：

- 决定在家中本地运行节点，还是在云上远程运行
- 为本地机器选择硬件，或选择远程托管提供商
- 准备和保护节点的操作系统
- 选择您的 Execution 和 Consensus 客户端
- 安装 Rocket Pool 的 Smartnode 堆栈
- 使用 Rocket Pool 网络创建新的信标链验证器
- 维护和更新您的节点
- 参与链上治理

两个指南都将向您展示如何在 Rocket Pool 测试网上免费练习，这样您就可以在不冒任何真实 ETH 风险的情况下感受一切，以及在您熟悉该过程后如何在 Ethereum 主网上质押您的真实 ETH。

准备好后，请根据您的兴趣从上面的选项中选择一个指南。
您也可以使用左侧的侧边栏随意浏览各个页面。
