# 通过 Ethereum 网络（第 1 层）上的分散交易所进行质押

提醒一下，您无需直接通过 Rocket Pool 交换 ETH 为 rETH 来享受去中心化质押的益处。
您所需要做的就是持有 rETH，因为它内在地增长价值 - 您如何获得它并不重要！

rETH 可在多个分散交易所进行交易，例如 [Balancer](https://docs.balancer.fi/) 和 [Uniswap](https://docs.uniswap.org/protocol/introduction)。

**Balancer** 池是一种特殊的池，称为 [metastable pool](https://docs.balancer.fi/concepts/pools/composable-stable.html)。
Metastable 池对于 rETH 等代币来说是理想的，因为 **它们尊重 Oracle DAO 报告的真实汇率**。

这意味着与此池交换将具有 **大幅较低的滑点** 和 **较低费用** 比传统分散交易所，所以您在买卖 rETH 时获得更好的交易。
其他分散交易所没有这种特性，所以您可能会发现他们的汇率略低。

无论您使用哪个池，他们都 **倾向于比通过官方 Rocket Pool 网站质押具有更低的汽油费**。
这使它们成为想要质押少量 ETH 的用户的最佳选择，尽管比较您在交易所获得的汇率与根据 Rocket Pool 的汇率总是一个好主意。

在以下示例中，我们将展示如何使用 Balancer 和 Uniswap 都交换 ETH 为 rETH。

::: warning 注意
目前此方法仅在主网上可用。
如果您想在 Hoodi 测试网上进行实践，您必须 [直接与 Rocket Pool 交换](./via-rp)。

**本指南的其余部分假设您在主网上。**
:::

此过程可以使用任何 web3 兼容钱包完成。
为了演示目的，我们将使用 [MetaMask](https://metamask.io/)，因为它是更受欢迎的钱包选择之一。

**如果您选择，您应该能够使用类似的步骤来跟进，只需在您的钱包中为每个步骤找到可比较的功能。**

## 设置 MetaMask

首先，如果您尚未安装，请安装 [MetaMask](https://metamask.io/)。
按照其网站上的说明安装扩展程序、创建帐户并登录。

接下来，使用浏览器工具栏中的图标打开 MetaMask 面板。
单击顶部工具栏中的 **网络下拉菜单**，并确保选择了 **Ethereum 主网**：

<img src="./images/mm_network_main.png" width="100%" height="auto" />

最后，将 rETH 代币添加到 MetaMask，以便您可以看到您的余额并访问它进行交易。
单击 **资产** 选项卡，然后单击 **添加代币**：

<img src="./images/mm_add_token.png" width="100%" height="auto" />

确保在此对话框中选择了 **自定义代币**。
在 **代币合约地址** 框中，输入以下值：

```
0xae78736Cd615f374D3085123A210448E74Fc6393
```

**代币符号** 应自动填充为 `rETH`，**精度小数位** 应自动填充为 `18`。

接受其余提示，然后您将在列表中看到 rETH 代币出现。

现在您在 MetaMask 中有了钱包地址，您需要将一些 ETH 转入其中。
**您需要从现有钱包或在交易所购买 ETH 来提供此。**

## 通过 Balancer 交换 rETH

现在您的钱包已设置并获得资金，请访问此 URL，这将打开 Balancer 并自动显示交换 ETH 为 rETH：

[https://app.balancer.fi/#/trade/ether/0xae78736Cd615f374D3085123A210448E74Fc6393](https://app.balancer.fi/#/trade/ether/0xae78736Cd615f374D3085123A210448E74Fc6393)

该页面最初将如下所示：

<img src="./images/balancer.png" width="100%" height="auto" style={{ margin: "0 auto" }} />

单击右上角的 `连接钱包` 按钮。
您将看到一个屏幕，询问您想使用哪个钱包提供商。
在本示例中，选择 **MetaMask**。

::: tip 注意
如果您不使用 MetaMask，其他提供商（如 WalletConnect）可能与您的钱包兼容。
:::

单击该按钮后，MetaMask 将弹出提示，要求您选择要使用的帐户。
假设您只有一个，您的帐户应该已经被选择。
单击 **下一个**。

然后它会询问您是否想给 Balancer 访问您的帐户地址的权限：

<img src="./images/bal_connect.png" width="100%" height="auto" style={{ margin: "0 auto" }} />

单击 **连接**，在短暂的暂停窗口后，您的钱包将连接到 Balancer。
您现在应该在屏幕的左上方看到其 ETH 余额。

现在，输入您想交换的 ETH 数量，Balancer 的 UI 将显示您将返回的 rETH 数量：

<img src="./images/bal_swap.png" width="100%" height="auto" style={{ margin: "0 auto" }} />

::: warning 注意
请注意，您收到的 rETH 数量将 **少于** 您投入的 ETH 数量。
这是正常的，因为 **rETH 的价值高于 ETH** 并随着时间的推移不断升值。
您可以在屏幕截图右侧突出显示的汇率中看到这一点。
:::

单击 **预览** 以查看交易详情，包括由于滑点可能收到的最少金额。
如果您满意，单击 **确认交易**。
这将向 MetaMask 发送请求，您可以在确认之前查看总交易成本：

<img src="./images/mm_gas.png" width="100%" height="auto" />

::: warning 注意
在此处突出显示的估计汽油费中 **密切注意**。
如果这是您总交易的很大一部分（这里是 17% 以上），那么在您打破质押奖励之前需要 **很长时间**。

如果是这样，您应该等待网络平均汽油价格下降，或者 [考虑改用第 2 层解决方案](./via-l2)。
:::

如果您对相对于您将获得的 rETH 总金额的汽油费感到满意，请单击 **确认** 来提交交易。
在短时间后，您应该看到来自 MetaMask 的通知，表明它已成功完成，您现在将在您的钱包中看到 rETH 代币。

就这样！您现在正在用 Rocket Pool 进行质押。您所需要做的就是保留您的 rETH，而您将自动获得质押奖励，因为 rETH 代币的价值增加。

## 通过 Uniswap 交换 rETH

虽然我们建议您通过 Balancer 进行质押，因为它的费用更低和更准确的 ETH-rETH 汇率，但如果需要，您也可以通过 Uniswap 交换 rETH。
访问此 URL，这将打开 Uniswap 并自动显示交换 ETH 为 rETH：

[https://app.uniswap.org/#/swap?inputCurrency=eth&outputCurrency=0xae78736cd615f374d3085123a210448e74fc6393](https://app.uniswap.org/#/swap?inputCurrency=eth&outputCurrency=0xae78736cd615f374d3085123a210448e74fc6393)
