# 용어 설명

Rocket Pool 문서에서 생소한 개념을 접하실 수 있습니다. 이 섹션은 쉬운 접근과 학습, 그리고 플러그인/테마 개발을 위해 문서에서 사용되는 일반적인 용어들을 정리합니다.

## APY - 연간 수익률 (Annual Percent Yield)

1년 기간 동안 계산된 수익(수익률)을 초기 투자 금액으로 나눈 값을 백분율로 표현한 것입니다.

## Bonding

_Insurance를 참조하세요._

## Contract, (Smart)

프로그래밍된 기능들을 수행하는 데 사용되는 Ethereum 프로그램(스마트 컨트랙트)입니다. Rocket Pool 스마트 컨트랙트에는 deposit pool, minipool validator 생성, 그리고 Ethereum 체인에서 다양한 토큰 상호작용을 발행하고 추적하는 것들이 포함됩니다.

## Client

각 블록의 모든 트랜잭션을 검증하여 네트워크를 안전하게 유지하고 데이터를 정확하게 유지하는 Ethereum 소프트웨어의 구현체입니다. _Execution과 Consensus client도 참조하세요._

## Custodial

사용자를 대신하여 전체 ETH staking 프로세스를 관리하고 개인 validator 키와 출금 키에 대한 "보관권"을 보유하는 중앙화된 서비스입니다. _Non-custodial도 참조하세요._

## DAO (Decentralized Autonomous Organization)

투명한 컴퓨터 프로그램으로 인코딩된 규칙으로 표현되고, 조직 구성원들에 의해 통제되며, 중앙 정부의 영향을 받지 않는 조직입니다.

## Deposit Pool

staking될 준비가 된 ETH의 양입니다. deposit pool은 일반 사용자가 ETH를 rETH로 교환할 때 자금이 조달됩니다. 이 ETH는 node operator 대기열의 node operator들과 짝지어지기를 기다리는 금액입니다.

## ETH

Ethereum 프로토콜에서 생성되고 사용되는 기본 암호화폐 코인입니다.

## Execution Client

execution layer를 실행하는 Ethereum 소프트웨어 클라이언트입니다.

## Consensus Client

consensus layer를 실행하는 Ethereum 소프트웨어 클라이언트입니다.

## eth2.0 (the Beaconchain)

2020년 12월에 출시되어 2022년 9월 proof-of-work eth1.0 합의 메커니즘을 대체한 Ethereum 2.0 네트워크 및 Ethereum 네트워크 업데이트입니다. 현재는 단순히 Beaconchain 또는 consensus layer로 알려진 더 이상 사용되지 않는 용어입니다.

## Insurance

node operator가 minipool validator를 시작할 때 예치해야 하는 RPL 토큰의 양으로, 프로토콜이 제공한 ETH의 가치에 대한 백분율로 표현됩니다. node operator가 stake한 RPL insurance는 node operator가 validator에서 프로토콜이 제공한 ETH보다 적은 금액으로 staking을 종료하는 경우(8 ETH minipool의 경우 24 ETH, 16 ETH minipool의 경우 16 ETH) 일반 사용자에게 배상하기 위한 보험 역할을 합니다.

## Minipool (Validator)

Rocket Pool 소프트웨어를 통해 자금이 조달되고 초기화된 validator입니다. deposit pool을 통해 일반 사용자가 기여한 ETH와 등록 프로세스 중 node operator가 기여한 ETH로 구성됩니다.

## Node

Ethereum 네트워크에서 node는 Ethereum 클라이언트 소프트웨어를 실행하는 장치(컴퓨터)입니다.

## Node Commission

node operator에게 수여되는 추가 ETH의 양입니다. 이것은 실질적으로 일반 staker들로부터 가져가는 "수수료"입니다; Rocket Pool에서는 이 수수료가 node operator들에게 주어집니다. Rocket Pool이 출시되었을 때, 이상적인 목표인 10%를 기준으로 5 - 20% 범위였으며, minipool 생성 시 각 minipool의 commission이 평생 고정됩니다. 이제 활성화되는 새로운 minipool들은 고정된 14% commission rate를 가집니다.

## Node Operator

Rocket Pool 소프트웨어, validator 클라이언트, 그리고 선택적으로 execution과 beaconchain 클라이언트도 실행할 node(컴퓨터)를 운영하여 Ethereum 네트워크를 보호하고자 하는 개인입니다. Node operator들은 Ethereum 네트워크의 solo-staker들보다 더 높은 비율의 ETH APY 보상을 얻습니다.

## Node Software

node operator들이 Rocket Pool 프로토콜과 상호작용할 수 있게 하는 프로그램들의 집합입니다.

## Node Wallet

새 node를 생성할 때 Rocket Pool에 의해 생성되는 Ethereum 지갑입니다. 이 지갑은 생성 중에 새 minipool에 자금을 제공하고 Rocket Pool의 스마트 컨트랙트와 상호작용하는 데 필요한 gas fee를 지불하는 데 사용됩니다.

## Non-Custodial

validator 설정 및 관리를 위한 간소화를 제공하지만 사용자 개인 validator 키와 출금 키를 보유하지 않는 서비스입니다. Rocket Pool은 non-custodial staking 서비스입니다.

## Oracle DAO (oDAO)

모든 oracle node operator들로 구성된 DAO입니다. _DAO도 참조하세요._

## Oracle Node

beacon chain으로부터의 정보(총 staking 보상 및 validator 종료 상태 등)를 execution layer로 다시 전달하여 Rocket Pool 스마트 컨트랙트가 올바르게 작동할 수 있도록 Rocket Pool 커뮤니티의 신뢰할 수 있는 구성원들이 실행하는 특별한 node입니다.

## Proof-of-Stake (PoS)

사용자들이 지갑에 있는 ETH를 stake하여 새로운 블록의 정확성에 투표하는 Ethereum에서 사용되는 합의 메커니즘입니다.

## Protocol DAO (pDAO)

RPL 토큰의 모든 보유자로 구성된 DAO입니다. _DAO도 참조하세요._

## Regular Node (Operator)

Oracle Node의 추가 책임을 갖지 않는 "일반" node operator들에게 주어지는 용어입니다.

## Regular Stakers

ETH 코인을 ETH의 기본 가치와 분리하지 않고 이자를 얻고자 하는 소매(일반) 사용자입니다. 일반 staker는 ETH 코인을 rETH 토큰으로 교환함으로써 Rocket Pool 플랫폼과 상호작용합니다. 그들의 ETH는 staking pool에 입력되며, Node Operator들이 새 minipool을 생성할 때 획득됩니다.

## rETH (Rocket Pool Staking Deposit Token)

일반 사용자의 staking pool 예치를 나타내는 토큰입니다. 사용자는 예치한 각 ETH 코인에 대해 동등한 가치의 rETH 토큰을 받습니다. Ethereum 네트워크에서 staking으로 이자가 발생함에 따라 rETH 토큰의 가치는 ETH에 대해 증가합니다. 이 토큰은 네트워크 내에서 잠글 필요가 없으며 사용자가 원하는 대로 거래, 판매 또는 보유할 수 있습니다.

## RPL (Rocket Pool Token)

**RPL — Rocket Pool Protocol Token**

RPL은 프로토콜의 거버넌스에 사용되고 Rocket Pool node에 stake할 수도 있는 주요 프로토콜 토큰입니다.

Node Operator 관점에서:

- Saturn 1부터, 투표 자격이 있는 stake된 RPL은 총 프로토콜 수익의 일부를 받게 됩니다
- RPL 인플레이션의 [대부분](https://rpips.rocketpool.net/RPIPs/RPIP-25)은 stake된 RPL을 기반으로 Node Operator들에게 보상으로 주어집니다; 이러한 보상은 Saturn 2와 함께 단계적으로 폐지될 계획입니다
- Saturn 0 minipool들(2024년 10월 28일 이후 생성)의 경우: Stake된 RPL은 최대 4%의 추가 보너스 commission을 제공합니다; 보너스 commission은 Saturn 1 출시 몇 달 후 단계적으로 폐지될 계획입니다.
- Stake는 거버넌스를 위한 투표권을 제공합니다

프로토콜 관점에서:

- RPL 인플레이션의 [소수](https://rpips.rocketpool.net/RPIPs/RPIP-25)는 pDAO에 자금을 제공합니다
- RPL 인플레이션의 [소수](https://rpips.rocketpool.net/RPIPs/RPIP-25)는 oDAO에 자금을 제공합니다
- 2차 담보(예: slashing에 대비)로 사용될 수 있습니다

## Stake

어떤 작업을 수행하기 위한 보증 담보로 예치되는 코인 또는 토큰의 양입니다. 개인은 ETH를 rETH로 교환함으로써 일반 사용자로서, 또는 Beaconchain PoS 합의 프로세스의 일부로 사용하기 위해 ETH를 예치함으로써 node operator로서 ETH를 stake할 수 있습니다. ETH를 stake하는 사람(예: staker)은 예치된 ETH에 대한 APY 또는 수익으로 보상을 받습니다.

## Staking Pool

staking pool을 통해 여러 ETH 코인 보유자들이 토큰을 결합하여 Ethereum validator를 실행하는 데 필요한 금액을 집단적으로 형성할 수 있습니다. Rocket Pool은 non-custodial 탈중앙화 방식을 사용하여 이러한 집단 pool들을 형성, 추적, 할당 및 보험 처리합니다.

## Validator

_Minipool을 참조하세요._
