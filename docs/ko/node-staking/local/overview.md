# 개요

이 섹션은 집에서 Rocket Pool 노드로 사용할 로컬 기기를 준비하는 프로세스를 이해하는 데 도움을 주기 위한 것입니다.

## 가이드

[Selecting Staking Hardware](./hardware)는 노드 운영을 위한 다양한 실행 가능한 하드웨어 구성을 보여주며, 여러분이 직접 선택할 때 영감을 얻을 수 있도록 도와줍니다.

[Preparing a PC, Mini-PC, or NUC](./prepare-pc)는 기기에 기본 운영 체제를 설치하고 구성하여 노드로서의 역할을 준비하는 기본 단계를 안내합니다.

[Preparing a Mac](./prepare-mac)은 동일한 내용이지만 macOS 시스템용입니다.

[Intro to Secure Shell (SSH)](../ssh)는 **SSH**를 사용하는 방법을 보여줍니다. SSH는 노드 기기에 모니터와 키보드를 직접 연결하지 않고 다른 기기(예: 노트북 또는 데스크톱)에서 노드 기기를 원격으로 안전하게 작업할 수 있는 인기 있는 원격 액세스 도구입니다.
