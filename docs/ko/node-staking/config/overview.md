# 개요

이 섹션에서는 Smartnode 스택을 구성하고 원하는 대로 모든 설정을 사용자 지정하는 방법을 배웁니다.
스택을 처음 실행하기 전에 구성해야 하지만, 실행 중일 때 언제든지 다시 돌아와서 다시 구성할 수 있습니다.

## 사전 요구 사항

Smartnode를 구성하기 전에 다음 사항을 확인하십시오.

- 운영 체제가 설치 및 구성된 노드 머신(또는 가상 머신)이 있어야 합니다
- [노드 보안 강화](../securing-your-node) 가이드를 통해 머신이 보안되어 있어야 합니다
- Smartnode가 설치되어 있어야 합니다

## 가이드

[Smartnode 스택 구성하기 (Docker / Hybrid 모드)](../config-docker)는 Docker 모드 또는 Hybrid 모드로 실행하려는 경우 사용할 가이드입니다.

[Smartnode 스택 구성하기 (Native 모드)](../config-native)는 Native 모드로 실행하려는 경우 사용할 가이드입니다.

[Docker 모드를 위한 고급 Smartnode 구성](../advanced-config)은 Docker 가이드의 보충 자료로, 관심이 있을 수 있는 추가 기능과 특징("Reverse Hybrid" 모드 또는 Docker 컨테이너 플래그 사용자 지정 등)을 보여줍니다.
