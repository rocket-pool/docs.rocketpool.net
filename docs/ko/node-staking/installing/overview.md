# 개요

이 섹션에서는 Rocket Pool Smartnode 스택을 노드 머신에 설치하는 방법을 배웁니다.
여러 가지 다른 설치 방식이 있으며, 어떤 것이 적합한지 알아보고 해당 설치 프로세스를 진행합니다.

## 전제 조건

이 섹션을 시작하기 전에 다음 사항을 확인하세요:

- 노드 머신(로컬 하드웨어 또는 가상 서버) 선택
- 운영 체제 설치
- [SSH](../ssh)와 같은 원격 제어 시스템 구성
- [Securing your Node](../securing-your-node) 가이드에 따라 보안 설정 완료

## 가이드

[Choosing your ETH Clients](../eth-clients)는 Execution Layer(이전 ETH1) 및 Consensus Layer(이전 ETH2)의 다양한 Ethereum client 옵션을 제시합니다.
노드를 실행하려면 각각 하나씩 필요하며, 이 섹션에서 실행할 클라이언트를 선택하는 데 도움이 됩니다.

[Selecting a Rocket Pool Mode](../install-modes)는 Rocket Pool Smartnode를 설치할 수 있는 다양한 모드를 보여주고 가장 적합한 모드를 선택하는 데 도움이 됩니다.

[Creating a Standard Rocket Pool Node with Docker](../docker)는 **Docker Mode** 및 **Hybrid Mode**의 설치 프로세스를 안내합니다.

[Creating a Native Rocket Pool Node without Docker](../native.mdx)는 **Native Mode**의 설치 프로세스를 안내합니다.
