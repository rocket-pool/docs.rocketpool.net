# Resumen

En esta sección, aprenderá cómo instalar el stack de Smartnode de Rocket Pool en su máquina de nodo.
Hay varios tipos diferentes de instalación; aprenderá cuál es el adecuado para usted y recorrerá el proceso de instalación para él.

## Requisitos Previos

Antes de sumergirse en esta sección, asegúrese de haber:

- Elegido una máquina de nodo (hardware local o servidor virtual)
- Configurado el Sistema Operativo en ella
- Configurado un sistema de control remoto como [SSH](../ssh)
- Asegurado siguiendo la guía [Asegurando su Nodo](../securing-your-node)

## Guías

[Eligiendo sus Clientes ETH](../eth-clients) presenta las diversas opciones para clientes de Ethereum - tanto en la Capa de Execution (anteriormente ETH1) como en la Capa de Consensus (anteriormente ETH2).
Necesitará uno de cada uno para ejecutar su nodo, y esta sección le ayudará a elegir cuáles le gustaría ejecutar.

[Seleccionando un Modo de Rocket Pool](../install-modes) le mostrará los diferentes modos en que se puede instalar el Smartnode de Rocket Pool y le ayudará a elegir el modo que sea mejor para usted.

[Creando un Nodo Estándar de Rocket Pool con Docker](../docker) le guiará a través del proceso de instalación para **Modo Docker** y **Modo Híbrido**.

[Creando un Nodo Nativo de Rocket Pool sin Docker](../native.mdx) le guiará a través del proceso de instalación para **Modo Nativo**.
