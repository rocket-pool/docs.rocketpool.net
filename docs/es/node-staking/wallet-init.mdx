# Creando una Nueva Cartera

Si los servicios de Smartnode están funcionando correctamente, el siguiente paso es configurar una cartera de Ethereum para tu nodo.
Esto es simplemente una dirección de la Capa de Ejecución que contendrá los fondos de tu nodo - la usará para enviar ETH a tu minipool cuando comiences el staking, pagar por gas durante varias transacciones y otras operaciones diversas.

Tienes la opción de crear una cartera completamente nueva desde cero, o importar/recuperar una cartera o dirección existente.
Por favor, elige la opción que prefieras en las pestañas a continuación.

<div className="p-3">
  <Tabs>
    <Tab label="Creando una Nueva Cartera">
      La forma más común de ejecutar un nodo es crear una nueva dirección de Ethereum dedicada al nodo.
      La CLI de Smartnode proporciona una manera fácil de hacer esto:

      ```shell
      rocketpool wallet init
      ```

      Primero se te pedirá una contraseña para proteger la clave privada de tu cartera.
      A continuación, se te presentará la **frase mnemotécnica única de 24 palabras** para tu nueva cartera.
      Esta es la **frase de recuperación** de tu cartera.
      Si alguna vez pierdes tu máquina, puedes usar esta frase para regenerar tu cartera y resucitar todos los validadores de Consensus conectados a ella.

      <p className="rspress-directive warning">
        <p className="rspress-directive-title">NOTA</p>
        Es **esencial** que escribas esta frase mnemotécnica porque esta es la única vez que se te mostrará, pero guárdala en un lugar seguro.
        Cualquiera con esta frase puede obtener el control de tu cartera.
      </p>

      Una vez que confirmes tu frase mnemotécnica, se te presentará la dirección única de Ethereum para la cartera de tu nodo.
    </Tab>
    <Tab label="Recuperando o Importando una Cartera Existente">
      Una vez que hayas terminado de ingresar la frase mnemotécnica, verás un mensaje de éxito como el siguiente:
      "La cartera del nodo se inicializó correctamente.
        Cuenta del nodo: {CadenaLargaDeLetrasYNúmeros}"
      Esta es la dirección pública a la que enviarás ETH en "Preparando tu Nodo para la Operación". También puedes verificarla escribiendo "rocketpool wallet status".

      Si ya tienes una cartera de nodo que creaste en una instalación anterior del Smartnode, o si deseas importar una dirección que creaste en otro lugar (como con **Allnodes**), por favor sigue las instrucciones en la guía [Importando una Cartera Existente para tu Nodo](./recovering-rp).

      Regresa aquí cuando hayas terminado.
    </Tab>

  </Tabs>
</div>

::: warning NOTA
A partir de la versión actual, Rocket Pool necesita tener acceso a la clave privada de tu cartera para realizar sus funciones automáticas.
**Esto significa que la clave privada existirá en un archivo en tu máquina.**
Si un atacante logra obtener acceso a tu máquina, podría obtener acceso a la cartera de tu nodo y robar todos los tokens que contenga.
Por favor, asegúrate de haber seguido la guía de seguridad en la sección [Asegurando tu Nodo](./securing-your-node) antes de continuar, y que te sientas cómodo con tu postura de seguridad.
:::
