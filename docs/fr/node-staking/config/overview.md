# Aperçu

Dans cette section, vous apprendrez à configurer la stack Smartnode et à personnaliser tous les paramètres selon vos préférences.
Vous devrez la configurer avant d'exécuter la stack pour la première fois, mais vous pouvez toujours revenir une fois qu'elle est en cours d'exécution et la reconfigurer à tout moment.

## Prérequis

Avant de configurer votre Smartnode, assurez-vous que vous avez :

- Une machine de nœud (ou machine virtuelle) avec le système d'exploitation installé et configuré
- Sécurisé la machine (via le guide [Sécuriser votre nœud](../securing-your-node))
- Le Smartnode installé dessus

## Guides

[Configuring the Smartnode Stack (Docker / Hybrid Mode)](../config-docker) est le guide à utiliser si vous souhaitez exécuter avec le mode Docker ou le mode hybride.

[Configuring the Smartnode Stack (Native Mode)](../config-native) est le guide à utiliser si vous souhaitez exécuter avec le mode natif.

[Advanced Smartnode Configuration for Docker Mode](../advanced-config) est un complément au guide Docker qui montre des capacités et fonctionnalités supplémentaires qui pourraient vous intéresser (comme le mode « Reverse Hybrid » ou la personnalisation des flags de conteneur Docker).
