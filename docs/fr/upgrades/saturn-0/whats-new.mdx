# La mise à jour Saturn 0

Bienvenue à Saturn 0 ! Cette mise à jour vise à améliorer la tokenomics du RPL à court terme pour débloquer la croissance du protocole. Si vous êtes un opérateur de nœud de retour ou prospectif, cette page servira d'introduction à certains changements nouveaux et excitants du protocole Rocket Pool.

## Minipools ETH uniquement dans Saturn 0

- L'exigence de RPL pour les nouveaux minipools appartient au passé ! Les minipools dans Saturn 0 ne nécessitent pas de stake RPL pour être déployés.
- L'exigence de RPL pour les nouveaux minipools appartient au passé ! Les minipools dans Saturn 0 ne nécessitent pas de stake RPL pour être déployés.
  Les opérateurs de nœuds peuvent désormais créer des minipools avec un coût initial de 8 ETH, qui comporteront une commission de 10%. De plus, un opérateur de nœud
  peut optionnellement staker du RPL pour se qualifier pour des récompenses supplémentaires et accéder au pouvoir de vote au sein de la [Protocol DAO](/fr/legacy/houston/pdao#the-protocol-dao-pdao).

- Les nœuds avec un stake RPL de n'importe quelle taille se qualifient pour les récompenses d'émission RPL. Cela débloque les récompenses RPL pour les opérateurs de nœuds en dessous du seuil de garantie de 10%
  qui n'étaient auparavant pas qualifiés pour les récompenses d'émission RPL.

- Cela signifie que les nœuds avec un stake RPL de 0-15% de l'ETH emprunté reçoivent l'APY maximum sur le RPL. Les nœuds gagnent toujours un rendement RPL au-delà de 15% de garantie, à un taux décroissant.
  N'hésitez pas à consulter comment l'émission de RPL au-delà de 15% d'ETH emprunté est définie dans [RPIP-30](https://rpips.rocketpool.net/RPIPs/RPIP-30#specification).

## Nouvelle structure de commission

Nous sommes ravis d'annoncer que les minipools Saturn 0 nouvellement lancés comporteront une structure de commission allant de 10% à 14%.

- Les nœuds sans RPL staké gagneront une commission de 10% (5% du contrat minipool, et 5% du boost de commission dynamique).
- Les nœuds avec du RPL staké reçoivent jusqu'à 4% de boost de commission dynamique supplémentaire pour du RPL staké évalué à >= 10% de l'ETH emprunté.

Notez que le boost de commission dynamique sera réduit à 0% lors du 5ème instantané de récompense après la sortie de Saturn 1 (ne laissant que la commission de 5% du contrat minipool). Cela sert à augmenter l'avantage de rendement de la migration vers les megapools.

::: warning AVERTISSEMENT
Votre nœud doit être inscrit au [Smoothing Pool](/fr/node-staking/fee-distrib-sp#the-smoothing-pool) pour bénéficier du boost de commission dynamique. Le Smart Node vous informera
de vous inscrire au Smoothing Pool lors du dépôt d'un minipool si vous n'êtes pas déjà inscrit. _Vos nouveaux minipools manqueront un rendement supplémentaire si votre nœud
n'est pas dans le Smoothing Pool !!_
:::

## Guide de l'opérateur de nœud

Si vous êtes intéressé par la gestion d'un nœud Rocket Pool pour créer vos propres validateurs Ethereum, consultez notre [guide de l'opérateur de nœud](/fr/node-staking/responsibilities).
Il s'agit d'un guide très complet qui couvre toutes les considérations impliquées dans l'exploitation de nœuds, notamment :

- Décider d'exécuter un nœud localement à la maison, ou à distance sur le cloud
- Sélectionner le matériel pour une machine locale, ou sélectionner un fournisseur d'hébergement distant
- Préparer et sécuriser le système d'exploitation de votre nœud
- Sélectionner vos clients d'Execution et de Consensus
- Installer la stack Smartnode de Rocket Pool
- Créer un nouveau validateur Beacon Chain en utilisant le réseau Rocket Pool
- Maintenir et mettre à jour votre nœud
- Participer à la gouvernance on-chain

Les deux guides vous montreront comment pratiquer gratuitement sur le testnet Rocket Pool afin que vous puissiez avoir une idée de tout sans risquer votre véritable ETH, et comment staker votre véritable ETH sur le mainnet Ethereum une fois que vous êtes à l'aise avec le processus.

Lorsque vous êtes prêt, sélectionnez un guide parmi les options ci-dessus en fonction de vos intérêts.
Vous êtes également libre d'explorer les différentes pages en utilisant la barre latérale à gauche.
