# Eine neue Wallet erstellen

Wenn die Smartnode-Dienste aktiv und betriebsbereit sind, ist der nächste Schritt die Einrichtung einer Ethereum-Wallet für Ihren Node.
Dies ist einfach eine Execution Layer-Adresse, die die Mittel Ihres Nodes verwahrt - sie wird verwendet, um ETH an Ihren Minipool zu senden, wenn Sie mit dem Staking beginnen, Gas für verschiedene Transaktionen zu bezahlen und andere verschiedene Operationen durchzuführen.

Sie haben die Möglichkeit, entweder eine völlig neue Wallet von Grund auf zu erstellen oder eine bestehende Wallet oder Adresse zu importieren bzw. wiederherzustellen.
Bitte wählen Sie aus den folgenden Registerkarten die gewünschte Option aus.

<div className="p-3">
  <Tabs>
    <Tab label="Eine neue Wallet erstellen">
      Die üblichste Methode zum Betreiben eines Nodes ist die Erstellung einer neuen Ethereum-Adresse, die ausschließlich für den Node bestimmt ist.
      Die Smartnode CLI bietet eine einfache Möglichkeit, dies zu tun:

      ```shell
      rocketpool wallet init
      ```

      Sie werden zunächst nach einem Passwort gefragt, um den privaten Schlüssel Ihrer Wallet zu schützen.
      Anschließend wird Ihnen die **einzigartige 24-Wort-Mnemonic** für Ihre neue Wallet präsentiert.
      Dies ist die **Wiederherstellungsphrase** für Ihre Wallet.
      Sollten Sie jemals Ihre Maschine verlieren, können Sie mit dieser Phrase Ihre Wallet wiederherstellen und alle damit verbundenen Consensus-Validatoren wiederherstellen.

      <p className="rspress-directive warning">
        <p className="rspress-directive-title">HINWEIS</p>
        Es ist **unerlässlich**, dass Sie diese Mnemonic aufschreiben, da sie Ihnen nur dieses eine Mal angezeigt wird, bewahren Sie sie jedoch an einem sicheren Ort auf.
        Jeder, der diese Phrase besitzt, kann die Kontrolle über Ihre Wallet erlangen.
      </p>

      Sobald Sie Ihre Mnemonic-Phrase bestätigt haben, wird Ihnen die eindeutige Ethereum-Adresse für Ihre Node-Wallet angezeigt.
    </Tab>
    <Tab label="Eine bestehende Wallet wiederherstellen oder importieren">
      Sobald Sie die Eingabe der Mnemonic-Phrase abgeschlossen haben, sehen Sie eine Erfolgsmeldung wie die folgende:
      „Die Node-Wallet wurde erfolgreich initialisiert.
        Node-Konto: {LangeZeichenfolgeAusBuchstabenUndZahlen}"
      Dies ist die öffentliche Adresse, an die Sie ETH senden werden, wie in „Vorbereitung Ihres Nodes für den Betrieb" beschrieben. Sie können sie auch durch Eingabe von „rocketpool wallet status" überprüfen.

      Wenn Sie bereits eine Node-Wallet haben, die Sie in einer früheren Installation des Smartnode erstellt haben, oder wenn Sie eine Adresse importieren möchten, die Sie anderswo erstellt haben (z. B. mit **Allnodes**), folgen Sie bitte den Anweisungen im Leitfaden [Importieren einer bestehenden Wallet für Ihren Node](./recovering-rp).

      Kehren Sie hierher zurück, wenn Sie fertig sind.
    </Tab>

  </Tabs>
</div>

::: warning HINWEIS
Ab dem aktuellen Build muss Rocket Pool Zugriff auf den privaten Schlüssel Ihrer Wallet haben, um seine automatischen Aufgaben ausführen zu können.
**Das bedeutet, dass der private Schlüssel in einer Datei auf Ihrer Maschine existiert.**
Sollte ein Angreifer Zugang zu Ihrer Maschine erlangen, könnte er Zugriff auf Ihre Node-Wallet erhalten und alle darin enthaltenen Token stehlen!
Bitte stellen Sie sicher, dass Sie den Sicherheitsleitfaden im Abschnitt [Sichern Ihres Nodes](./securing-your-node) befolgt haben, bevor Sie fortfahren, und mit Ihrer Sicherheitslage zufrieden sind.
:::
