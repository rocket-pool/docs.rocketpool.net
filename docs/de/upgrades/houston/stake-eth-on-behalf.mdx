# ETH im Namen eines Nodes staken

RPIP-32 erlaubt es einem Konto, [ETH im Namen](https://rpips.rocketpool.net/RPIPs/RPIP-32) eines Rocket Pool-Nodes zu staken, der im Protokoll registriert ist. Dies unterstützt eine Vielzahl von Situationen, in denen der Node-Operator das ETH nicht direkt bereitstellt:

- Verbesserte Sicherheit für Node-Operatoren, da sie direkt von ihrem Hardware-Wallet staken können, ohne Mittel vorher zum Node übertragen zu müssen.
- Staking-as-a-Service-Anbieter, bei denen das Sorgerecht für Mittel von einem vertrauenswürdigen Verwahrer verwaltet wird.
- Protokollintegrationen, bei denen das Sorgerecht für Mittel von Smart Contracts verwaltet wird.
- DAOs oder Organisationen, bei denen das Sorgerecht für Mittel von einer Treasury verwaltet wird.

Während das primäre Ziel dieser Funktion darin besteht, Einzeleinzahlungsszenarien zu erleichtern, ist es erwähnenswert, dass mehrere unabhängige Einzahler diese Fähigkeit auch nutzen können, indem sie Smart Contracts darüber legen. Rocket Pool führte auch die Möglichkeit ein, [RPL im Namen](../../node-staking/create-validator#whitelisting-an-address-to-stake-on-behalf) eines Nodes zu staken, bereits in unserem vorherigen Atlas-Release.

### Voraussetzung

Stellen Sie sicher, dass Sie die [primäre Abhebungsadresse](../../node-staking/prepare-node#setting-your-primary-withdrawal-address) des Nodes, den Sie finanzieren möchten, ordnungsgemäß konfiguriert haben. Folgen Sie dem obigen Link für eine Anleitung, wie Abhebungsadressen nach Houston funktionieren.

::: tip HINWEIS
Überspringen Sie diesen Schritt nicht! Alle ETH-Bonds/Belohnungen werden an die primäre Abhebungsadresse abgehoben. Mit anderen Worten, die Adresse, die ETH für einen Node bereitstellt, sollte als primäre Abhebungsadresse verwendet werden! Bitte stellen Sie sicher, dass die primäre Abhebungsadresse des Nodes auf ein Wallet gesetzt ist, über das Sie die Kontrolle haben.
:::

Der Prozess ist für das Hoodi Testnet und Mainnet leicht unterschiedlich, wählen Sie also das Netzwerk aus, an dem Sie interessiert sind, aus den Tabs unten.

Sobald Sie auf der Website sind, klicken Sie auf die Schaltfläche **connect wallet**. Bitte lesen Sie die Terms of Service & Privacy Policy durch und akzeptieren Sie sie, dies ermöglicht verschiedene Möglichkeiten zur Verbindung, klicken Sie dann auf **connect metamask**.

MetaMask fordert Sie auf, ein Konto auszuwählen, um sich mit der Website zu verbinden.
Wählen Sie eines aus, bestätigen Sie einige Berechtigungen. Sie sehen eine Übersicht Ihrer Guthaben, indem Sie auf das Wallet-Symbol oben rechts im Fenster klicken.

Von hier aus möchten Sie die Adresse des Nodes eingeben, für den Sie ETH im Namen hinterlegen möchten. Die Seite sollte so aussehen:

<img src="./images/enter_node_address.png" width="100%" height="auto" />

Fügen Sie Ihre gewünschte Adresse in dieses Feld ein und klicken Sie auf Find. In diesem Beispiel verwenden wir `0xd3e317806456102d19db283b4c2201f0ef41a296`

<img src="./images/eth_on_behalf.png" width="100%" height="auto" />

Geben Sie im ersten Feld die Menge an ETH ein, die Sie im Namen des Nodes hinterlegen möchten. Das zweite Feld zeigt an, für welche Adresse Sie ETH im Namen hinterlegen. Bitte stellen Sie sicher, dass Sie die richtige Adresse und Einzahlung eingegeben haben. Sobald Sie zufrieden sind, klicken Sie auf Deposit und MetaMask öffnet ein Fenster, das Sie auffordert, Ihre Transaktion zu bestätigen.

Sobald Sie sie bestätigen, beginnt die Transaktion.
Wenn sie in einen Block aufgenommen und der Chain hinzugefügt wurde, kann Ihr Delegat den Befehl `rocketpool node status` im Smartnode verwenden, um das neue Guthaben zu sehen. Dieses Guthaben kann auch über die Rocket Pool-Staking-Website angezeigt werden.

```
The node has 0.000000 ETH in its credit balance and 8.000000 ETH staked on its behalf. 8.000000 can be used to make new minipools.
```

::: warning HINWEIS
Wenn ein Node-Operator Ihr ETH in einen Minipool einzahlt, können Sie deren Minipool ohne Zugriff auf ihre Abhebungsdaten nicht beenden, also beachten Sie dies! Sie können die Rocket Pool-Staking-Website verwenden, um ETH, das im Namen eines Nodes gestakt wurde, abzuheben, wenn es nicht in einem Minipool verwendet wird.
:::

Nach Eingabe einer Adresse bringt Sie das Klicken auf "Find" zum nächsten Menü, wo Sie ETH abheben können, das Sie im Namen dieses Nodes gestakt haben.

<img src="./images/withdraw_eth_on_behalf.png" width="100%" height="auto" />

Sie können oben rechts sehen, wie viel ETH zur Abhebung verfügbar ist. Nachdem Sie Ihre gewünschte Menge eingegeben und die Node-Adresse bestätigt haben, klicken Sie auf die orangefarbene Schaltfläche **Withdraw ETH**, um die Transaktion an das Netzwerk zu senden.

Sobald die Transaktion bestätigt ist, sehen Sie die richtigen Guthaben in Ihrem Konto!
