# Saturn 0アップグレード

Saturn 0へようこそ! このアップグレードは、プロトコルの成長を促進するために、短期的にRPLトークノミクスを改善することを目的としています。既存または今後のNode Operatorの方は、このページでRocket Poolプロトコルのいくつかの新しく刺激的な変更についてご紹介します。

## Saturn 0でのETHのみのminipool

- 新しいminipoolのRPL要件は過去のものになりました! Saturn 0のminipoolをデプロイするためにRPLのステーキングは必要ありません。
- 新しいminipoolのRPL要件は過去のものになりました! Saturn 0のminipoolをデプロイするためにRPLのステーキングは必要ありません。
  Node Operatorは8 ETHの初期費用でminipoolを作成できるようになり、10%の手数料が適用されます。さらに、Node Operatorは
  オプションでRPLをステーキングして、追加の報酬を得たり、[Protocol DAO](/ja/legacy/houston/pdao#the-protocol-dao-pdao)内での投票権にアクセスしたりすることができます。

- 任意のサイズのRPLステークを持つノードは、RPL発行報酬の対象となります。これにより、以前はRPL発行報酬の対象外だった10%担保の閾値未満のNode Operatorに
  RPL報酬が解放されます。

- これは、借入ETHの0-15%のRPLステークを持つノードが、RPLの最大APYを受け取ることを意味します。ノードは15%を超える担保でもRPL利回りを獲得しますが、利率は低下します。
  借入ETHの15%を超えるRPL発行がどのように定義されているかについては、[RPIP-30](https://rpips.rocketpool.net/RPIPs/RPIP-30#specification)をご覧ください。

## 新しい手数料体系

新しく開始されるSaturn 0のminipoolは、10%から14%の範囲の手数料体系を特徴としています。

- ステーキングされたRPLがないノードは10%の手数料を獲得します(minipoolコントラクトから5%、動的手数料ブーストから5%)。
- ステーキングされたRPLを持つノードは、借入ETHの10%以上の価値があるRPLをステーキングすることで、最大4%の追加の動的手数料ブーストを受け取ります。

動的手数料ブーストは、Saturn 1のリリース後の5回目の報酬スナップショット時に0%に減少することに注意してください(5%のminipoolコントラクト手数料のみが残ります)。これは、megapoolへの移行の利回り優位性を高めるためです。

::: warning 警告
動的手数料ブーストの恩恵を受けるには、ノードが[Smoothing Pool](/ja/node-staking/fee-distrib-sp#the-smoothing-pool)にオプトインする必要があります。Smart Nodeは、
まだオプトインしていない場合、minipoolをデポジットするときにSmoothing Poolにオプトインするように通知します。_ノードがSmoothing Poolに入っていない場合、新しいminipoolは追加の利回りを逃してしまいます!!_
:::

## Node Operatorガイド

Rocket Poolノードを実行して独自のEthereumバリデーターを作成することに興味がある場合は、[Node Operatorガイド](/ja/node-staking/responsibilities)をご覧ください。
これは、以下を含むノード運用に関わるすべての考慮事項を網羅した非常に包括的なガイドです。

- ノードを自宅でローカルに実行するか、クラウドでリモートに実行するかの決定
- ローカルマシンのハードウェアの選択、またはリモートホスティングプロバイダーの選択
- ノードのオペレーティングシステムの準備と保護
- ExecutionクライアントとConsensusクライアントの選択
- Rocket PoolのSmartnodeスタックのインストール
- Rocket Poolネットワークを使用した新しいBeacon Chainバリデーターの作成
- ノードのメンテナンスと更新
- オンチェーンガバナンスへの参加

両方のガイドでは、実際のETHをリスクにさらすことなく、すべてを無料でRocket Poolテストネットで練習する方法と、プロセスに慣れたらEthereumメインネット上で実際のETHをステーキングする方法を紹介しています。

準備ができたら、興味に基づいて上記のオプションからガイドを選択してください。
左側のサイドバーを使用して、さまざまなページを自由に探索することもできます。
