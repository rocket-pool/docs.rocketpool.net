# A Atualização Saturn 0

Bem-vindo ao Saturn 0! Esta atualização visa melhorar a tokenômica do RPL no curto prazo para desbloquear o crescimento do protocolo. Se você é um operador de nó que está retornando ou em potencial, esta página servirá como uma introdução a algumas mudanças novas e emocionantes no protocolo Rocket Pool.

## Minipools somente com ETH no Saturn 0

- O requisito de RPL para novos minipools é coisa do passado! Os minipools no Saturn 0 não requerem staking de RPL para serem implantados.
- O requisito de RPL para novos minipools é coisa do passado! Os minipools no Saturn 0 não requerem staking de RPL para serem implantados.
  Os operadores de nós agora podem criar minipools com um custo inicial de 8 ETH, que terá uma comissão de 10%. Além disso, um operador de nó pode opcionalmente fazer staking de RPL para se qualificar para recompensas adicionais e acessar poder de voto dentro do [Protocol DAO](/pt/legacy/houston/pdao#the-protocol-dao-pdao).

- Nós com uma quantidade de RPL em staking de qualquer tamanho se qualificam para recompensas de emissão de RPL. Isso desbloqueia recompensas de RPL para Operadores de Nó abaixo do limite de garantia de 10%
  que anteriormente não se qualificavam para recompensas de emissão de RPL.

- Isso significa que nós com uma quantidade de RPL em staking de 0-15% do ETH emprestado recebem o APY máximo em RPL. Os nós ainda ganham rendimento de RPL além de 15% de garantia, a uma taxa decrescente.
  Sinta-se à vontade para dar uma olhada em como a emissão de RPL além de 15% do ETH emprestado é definida em [RPIP-30](https://rpips.rocketpool.net/RPIPs/RPIP-30#specification).

## Nova Estrutura de Comissão

Estamos animados em anunciar que os minipools Saturn 0 recém-lançados apresentarão uma estrutura de comissão variando de 10% a 14%.

- Nós sem nenhum RPL em staking ganharão comissão de 10% (5% do contrato do minipool e 5% do impulso de comissão dinâmica).
- Nós com RPL em staking recebem até um impulso de comissão dinâmica adicional de 4% para RPL em staking avaliado em >= 10% do ETH emprestado.

Observe que o impulso de comissão dinâmica será reduzido para 0% no 5º snapshot de recompensa após o lançamento do Saturn 1 (deixando apenas a comissão de contrato do minipool de 5%). Isso serve para aumentar a vantagem de rendimento da migração para megapools.

::: warning AVISO
Seu nó deve estar inscrito no [Smoothing Pool](/pt/node-staking/fee-distrib-sp#the-smoothing-pool) para se beneficiar do impulso de comissão dinâmica. O Smart Node informará você
para aderir ao Smoothing Pool ao depositar um minipool se você ainda não estiver inscrito. _Seus novos minipools perderão rendimento adicional se seu nó
não estiver no Smoothing Pool!!_
:::

## Guia do Operador de Nó

Se você estiver interessado em executar um nó Rocket Pool para criar seus próprios validators Ethereum, dê uma olhada no nosso [guia do Operador de Nó](/pt/node-staking/responsibilities).
Este é um guia muito abrangente que aborda todas as considerações envolvidas na operação de nós, incluindo:

- Decidir executar um nó localmente em casa ou remotamente na nuvem
- Selecionar o hardware para uma máquina local ou selecionar um provedor de hospedagem remota
- Preparar e proteger o sistema operacional do seu nó
- Selecionar seus clientes de Execução e Consenso
- Instalar a pilha Smartnode da Rocket Pool
- Criar um novo validator da Beacon Chain usando a rede Rocket Pool
- Manter e atualizar seu nó
- Participar da governança on-chain

Ambos os guias mostrarão como praticar gratuitamente na testnet da Rocket Pool para que você possa ter uma noção de tudo sem arriscar nenhum de seu ETH real, e como fazer staking de seu ETH real na mainnet Ethereum quando estiver confortável com o processo.

Quando estiver pronto, selecione um guia nas opções acima com base em seus interesses.
Você também é livre para explorar as várias páginas usando a barra lateral à esquerda.
