# Создание нового кошелька

Если службы Smartnode запущены и работают, следующий шаг — настроить Ethereum кошелек для вашей ноды.
Это просто адрес уровня исполнения, который будет хранить средства вашей ноды — он будет использоваться для отправки ETH в ваш minipool при начале стейкинга, оплаты газа во время различных транзакций и других операций.

У вас есть возможность либо создать совершенно новый кошелек с нуля, либо импортировать / восстановить существующий кошелек или адрес.
Пожалуйста, выберите нужный вариант из вкладок ниже.

::: warning ПРИМЕЧАНИЕ
В текущей версии Rocket Pool необходим доступ к приватному ключу вашего кошелька для выполнения автоматических обязанностей.
**Это означает, что приватный ключ будет существовать в файле на вашей машине.**
Если злоумышленник получит доступ к вашей машине, он может получить доступ к кошельку вашей ноды и украсть все содержащиеся в нем токены!
Пожалуйста, убедитесь, что вы следовали руководству по безопасности в разделе [Защита вашей ноды](./securing-your-node) перед тем, как продолжить, и уверены в своей безопасности.
:::
